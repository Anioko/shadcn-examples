## Getting Started

> This project is Next.js project.

Clone the project:

```bash
git clone git@github.com:shadcn-examples/shadcn-examples.git
```

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
